# ⏰ Digital Clock – Live Time & Date App

A simple and elegant **Digital Clock Web App** built using **HTML, CSS, and JavaScript**.
It displays the **current time and date in real-time**, updating every second.

---

## ✨ Features

* ⏱️ Live digital clock (HH:MM:SS format)
* 📅 Current date display
* ⚡ Real-time updates every second
* 🎨 Clean and minimal UI
* 📱 Responsive design

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ⚙️ How It Works

* JavaScript `Date()` object is used to fetch current time
* `setInterval()` updates the clock every second
* DOM manipulation updates the UI dynamically
